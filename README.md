# moodmansrejections

[![Deploy static site to Pages](https://github.com/ArmyaAli/moodmansrejections/actions/workflows/deploy-pages.yml/badge.svg)](https://github.com/ArmyaAli/moodmansrejections/actions/workflows/deploy-pages.yml)

Simple static site documenting Mehmood's legendary Discord call rejections.

## Deployment

This repo uses GitHub Actions to deploy `index.html` to GitHub Pages on every push to `main`.

After your first push, verify:

1. `Settings -> Pages -> Source` is set to `GitHub Actions`.
2. The workflow run named `Deploy static site to Pages` succeeds.
3. Your live site appears at `https://armyaali.github.io/moodmansrejections/`.
